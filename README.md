# ExplainEDU: AI for Transparent Student Analysis

## Project Overview  
ExplainEDU is a data analytics project aimed at exploring and interpreting student perceptions using advanced statistical and machine learning techniques. The project emphasizes explainable AI and data balancing to ensure fairness and transparency in analyzing survey data. All analysis and modeling were performed in Google Colab, utilizing Python for data processing and machine learning workflows.

## Key Features  
- **Data Analytics**: Conducted comprehensive exploratory data analysis (EDA) to identify patterns and trends in student feedback.  
- **Explainable AI**: Used SHAP (SHapley Additive exPlanations) to interpret feature importance and provide transparent insights into model predictions.  
- **Class Imbalance Handling**: Applied SMOTE (Synthetic Minority Oversampling Technique) to address imbalanced classes in survey data.  
- **Statistical Learning**: Performed regression analysis, hypothesis testing, and statistical modeling to uncover key factors influencing student perceptions.  
- **Machine Learning Models**: Built and evaluated models like Random Forest and XGBoost to predict satisfaction levels and analyze feature contributions.  

## Technologies Used  
### Programming Languages and Platforms  
- **Python**: Used extensively for data cleaning, analytics, and modeling.  
- **Google Colab**: Primary environment for collaborative development and analysis.  

### Libraries and Frameworks  
- **Data Analytics**: Pandas, NumPy  
- **Visualization**: Matplotlib, Seaborn  
- **Machine Learning**: Scikit-learn, XGBoost  
- **Explainable AI**: SHAP  
- **Imbalanced Data Handling**: imbalanced-learn (for SMOTE)  

## Implementation Highlights  
- **Data Preprocessing**:  
  - Cleaned and transformed survey data to handle missing values and outliers.  
  - Balanced class distributions using SMOTE to enhance model performance.  
- **Exploratory Data Analysis (EDA)**:  
  - Analyzed trends and distributions in student perceptions.  
  - Visualized key insights using Matplotlib and Seaborn.  
- **Modeling and Explainability**:  
  - Trained machine learning models like Random Forest and XGBoost to predict satisfaction scores.  
  - Used SHAP to interpret model outputs and identify key features influencing predictions.  
- **Statistical Insights**:  
  - Conducted hypothesis testing and regression analysis to validate findings.
